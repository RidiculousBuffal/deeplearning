# 安装torch
1. 前往[https://pytorch.org/get-started/locally/](https://pytorch.org/get-started/locally/) 选择合适的版本
2. 安装
```bash
uv pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu126 
```
